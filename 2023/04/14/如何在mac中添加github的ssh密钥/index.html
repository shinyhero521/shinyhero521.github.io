如何在mac中添加github的ssh密钥

在Mac中添加Github的SSH密钥可以按照以下步骤进行：

  1. 输入以下命令，将公钥复制到剪贴板中：
  1. 登录您的Github帐户，单击右上角的个人资料图标，然后选择“Settings”。
  2. 在左侧导航菜单中，单击“SSH and GPG keys”。
  3. 单击“New SSH key”按钮。
  4. 在“Title”字段中输入一个描述性名称，例如“我的Mac电脑”。
  5. 将刚才复制到剪贴板中的公钥粘贴到“Key”字段中。
  6. 单击“Add SSH key”按钮。

提示错误

这个错误提示表明您尝试连接的远程主机（可能是Github）的SSH密钥已更改，或者出现了中间人攻击的情况。因此，系统无法验证主机的真实性，导致了此错误。

为了解决此问题，您需要更新您本地计算机上的known_hosts文件，该文件存储与之前连接过的所有SSH服务器的公钥指纹。在这种情况下，您需要删除旧的密钥，并将新的密钥添加到known_hosts文件中。
